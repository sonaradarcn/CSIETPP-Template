## 前言

本仓库为大学生创新创业训练计划相关材料模板仓库，项目以Parkinglot-Navigation-Robot-V2为例（该项目为国家级大学生创新创业训练计划立项项目，目前仓库仅上传了V3版本。由于V3版本具备V2版本所有功能，且V2版本存在较多未修复问题，故不提供V2版本），示范如何编写创新创业训练计划项目材料。

项目代码详情请见[Parkinglot-Navigation-Robot-V3](https://github.com/sonaradarcn/Parkinglot-Navigation-Robot-V3)，本仓库仅提供大学生创新创业训练计划相关材料，所有材料仅进行匿名化处理并抹除相关敏感信息，其余信息未进行修改。

## 仓库文件介绍

- Application Form.doc

  大学生创新训练项目计划申请书

- Inspection Report - 1.doc

  项目第一季度检查报告

- Inspection Report - 2.doc

  项目第二季度检查报告（中期检查报告）

- Inspection Report - 2.pptx

  项目第二季度检查演示文档

- Inspection Report - 3.doc

  项目第三季度检查报告

由于本人开发电脑丢失数据，关于结项报告材料信息缺失。本人建议可以参考 Parkinglot-Navigation-Robot-V3 项目的项目报告。

## 结尾

本人希望通过公开上述信息，为相关申报者提供一定参考，指出项目立项的实际难度未必如想象中高，其关键更在于项目构想是否具备足够的新颖性与创新性。

> 此外，本人特别说明，就本人所在院校而言，各类季度检查报告多集中于2024年4月至5月期间集中撰写与提交。此前向学校相关部门咨询时，未获得关于此时间安排的进一步解释。因此，若项目已立项但长时间未接到提交材料的通知，建议无需过度焦虑，可能属于正常安排。

> 关于项目资助经费，本人所在学校对每个国家级立项项目核定5000元支持资金。然而，该笔经费实际拨付至项目挂名指导教师的科研账户，需通过合规发票报销流程方可支取。若申报者及其团队成员已垫付较多发票却未收到相应经费，建议认真审视经费申领与报销环节的具体情况（仅限自命题项目而非使用指导教师项目申报，对于使用指导教师项目申报应当合理沟通报销事宜）。
